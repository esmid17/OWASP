# OWASP Top 10 Demo

Demo frontend estática y responsiva para mostrar ejemplos funcionales de las vulnerabilidades del OWASP Top 10.

## Cómo usar

1. Ejecuta `npm install` una sola vez para instalar el servidor local.
2. Ejecuta `npm start` para levantar la app.
3. Abre la URL que te muestre la terminal.

## Incluye

- Tarjetas navegables para cada riesgo.
- Ejemplos interactivos y locales.
- Diseño adaptado a móvil.
